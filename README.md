Journey is a deep, text-based rogue-lite tower RPG built with Jetpack Compose. Players ascend the mysterious Cosmic Spires, making moral decisions that shift their **Momentum** between the **Sanctum** and the **Covenant**, while engaging in tactical turn-based combat.

## 🌌 Key Features

- **Dynamic Tower Climbing:** 100 floors of procedural and handcrafted challenges.
- **Momentum System:** Your choices organically shift your alignment, affecting dialogue and gameplay.
- **Tactical Combat:** A revamped combat system with intents, status effects, and strategic resource management.
- **Dark Fantasy Narrative:** A rich story with dual-language support (TR/EN).
- **Procedural Blueprints:** Every run offers a unique path through the tower.

## 🛠️ Project Structure

- `ui/screens/`: Individual screens for Tower, Character, Quests, Journal, etc.
- `ui/theme/`: Centralized design system (Colors, Typography, Dimens).
- `data/engine/`: Core game logic, blueprint systems, and localization.
- `data/model/`: Room database entities and data structures.

## 🚀 Getting Started

**Prerequisites:** [Android Studio Jellyfish+](https://developer.android.com/studio)

1. **Clone & Open:** Open the project in Android Studio.
2. **Environment Setup:** 
   - Create a `.env` file in the root directory.
   - Add `GEMINI_API_KEY=your_key_here` (for AI-driven features).
3. **Build:** Sync Gradle and build the project.
4. **Run:** Deploy to an emulator or physical device.
